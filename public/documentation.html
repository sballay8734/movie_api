Movie API

This project simply retrieves data from a custom build movie API and formats the responses in an easy to read and use format.

Base URL

www.something.com

URL Endpoints

Logic URL HTTP Method Request Body Data Format Response Body Data Format
Return a list of all movies /movies GET None A list of movie objects
Return data of a single movie /movies/[movieName] GET None A JSON object holding data about chosen movie
Return description of given genre /movies/genres/[genreName] GET None A list of all movies containing selected genre
Return description of director /movies/directors/[directorName] GET None A JSON object containing director info
Logic URL HTTP Method Request Body Data Format Response Body Data Format
Register a new user /users POST A JSON object holding data about new user A JSON object holding data about the added user
Update user info /users/[id] PUT A JSON object containing data to change A JSON object containing changes made
Add movie to favorites /users/[userName]/favorites POST A JSON object holding data about movie A text message indicating the change
Remove movie from favorites /users/[userName]/favorites DELETE A JSON object holding data about movie A text message indicating the change
Deregister account/Remove email /users/[userName]/deregister DELETE None A text message indicating the change
